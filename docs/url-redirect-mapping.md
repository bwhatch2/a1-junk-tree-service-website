# URL Redirect Mapping: Old Site to New Site

This document maps every known URL from the old site (`a1junkremovalofomaha.com`) to its corresponding URL on the new site (`a1junkremovalandtreeservice.com`). These 301 redirects will be added to the Netlify `_redirects` file when the old domain is pointed to Netlify.

**Rule: No old URL should ever return a 404. Every URL gets a redirect.**

## Important Notes

1. The redirect rules MUST be placed ABOVE the `/*` catch-all SPA rule in the `_redirects` file. Netlify processes rules top-to-bottom and uses the first match.
2. These redirects only take effect when the old domain is added as a domain alias in Netlify.
3. After adding these redirects, submit the old domain in Google Search Console and use the "Change of Address" tool.
4. Keep the old domain registered and redirecting for at least 1-2 years.
